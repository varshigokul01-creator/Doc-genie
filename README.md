# Doc-genie
Doc-Genie is an automated Python docstring generator designed to help developers write professional and structured documentation for their Python functions. Writing docstrings manually can be time-consuming and inconsistent. This tool simplifies the process by automatically analyzing Python functions and generating standardized docstrings.

Doc-Genie – How it Works 

The user enters text in the input box,The program processes the text using Python.
The system converts the text into a document file.
It creates files like PDF using ReportLab or TXT.
Finally, the user can download the generated file.
To automatically generate professional docstrings for Python functions.
To support multiple documentation styles such as Google style and NumPy style.
To improve code readability and maintainability.
To save developers' time and effort.

Tools:

Python – Used as the main programming language to run the program.
Gradio – Used to create a simple user interface with input boxes and buttons.
ReportLab – Used to convert text into PDF files.
Google Colab – Used to write and run the code online.

Why it is useful:

Saves time in creating documents
Easy to generate PDF or TXT files
Simple and user-friendly system
